Politique de confidentialité

Dernière mise à jour : 15 janvier 2025

1. Introduction

Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons vos données personnelles lorsque vous utilisez l'application d'intégration TikTok Live avec Minecraft.

Nous nous engageons à protéger votre vie privée et à respecter le Règlement Général sur la Protection des Données (RGPD) et les lois applicables sur la protection des données.

2. Collecte de données

Cette application collecte uniquement les données nécessaires à l'intégration TikTok Live :

  • Événements de live : Cadeaux, donations, likes, commentaires, abonnements reçus pendant vos lives TikTok
  • Informations de base du compte : Nom d'utilisateur TikTok et ID unique (pour l'authentification via l'API TikTok)
  • Données de connexion : Informations nécessaires pour établir la connexion WebSocket entre le serveur et Minecraft

Nous ne collectons PAS :

  • Votre mot de passe TikTok
  • Vos informations de paiement
  • Votre adresse email (sauf si fournie via l'API TikTok)
  • Vos données de localisation

3. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Recevoir les événements en temps réel pendant vos lives TikTok via l'API TikTok Live
  • Transmettre ces événements à votre instance Minecraft via WebSocket
  • Déclencher les actions correspondantes dans le jeu (spawn d'items, effets, messages)
  • Authentifier votre accès à l'API TikTok

Les données ne sont utilisées à aucune autre fin et ne sont pas partagées avec des tiers.

4. Stockage des données

Les données sont traitées en temps réel et ne sont pas stockées de manière permanente sur nos serveurs. Aucune base de données n'est utilisée pour conserver vos données personnelles.

Les seules données conservées sont :

  • Les tokens d'accès TikTok (nécessaires pour l'authentification API) - stockés localement dans votre fichier .env
  • Les logs de connexion (pour le débogage) - supprimés automatiquement après 7 jours

5. Partage des données

Nous ne partageons, ne vendons ni ne louons vos données personnelles à des tiers. Les données sont utilisées exclusivement pour le fonctionnement de l'application.

Vos données peuvent être transmises à :

  • TikTok : Pour l'authentification et la réception des événements (conformément à leur politique de confidentialité)
  • Votre instance locale : Les données sont transmises uniquement à votre serveur local et à votre jeu Minecraft

6. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données contre tout accès non autorisé, altération, divulgation ou destruction :

  • Chiffrement des communications via HTTPS/WebSocket sécurisé
  • Stockage sécurisé des tokens d'accès (fichier .env non versionné)
  • Validation des webhooks TikTok pour éviter les requêtes frauduleuses

7. Vos droits (RGPD)

Conformément au RGPD et aux lois sur la protection des données, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données si elles sont inexactes
  • Supprimer vos données (droit à l'oubli)
  • Vous opposer au traitement de vos données
  • Portabilité de vos données
  • Limiter le traitement de vos données

Pour exercer ces droits, veuillez ouvrir une issue sur le repository GitHub.

8. Cookies

Cette application n'utilise pas de cookies pour le suivi des utilisateurs. Aucun cookie n'est déposé sur votre navigateur.

9. Données des mineurs

Cette application n'est pas destinée aux utilisateurs de moins de 13 ans. Si vous avez moins de 13 ans, veuillez ne pas utiliser cette application. Si nous apprenons qu'un utilisateur de moins de 13 ans utilise l'application, nous supprimerons ses données.

10. Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une mise à jour de la date de "Dernière mise à jour".

Nous vous encourageons à consulter régulièrement cette page pour rester informé de la façon dont nous protégeons vos données.

11. Contact

Pour toute question concernant cette politique de confidentialité ou pour exercer vos droits, veuillez :

  • Ouvrir une issue sur le repository GitHub
  • Consulter les Conditions d'utilisation
