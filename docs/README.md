# Documentation GitHub Pages

Ce dossier contient les pages web accessibles via GitHub Pages pour l'application TikTok Minecraft Integration.

## Pages disponibles

- **index.html** - Page d'accueil
- **terms.html** - Conditions d'utilisation
- **privacy.html** - Politique de confidentialité

## URLs GitHub Pages

Une fois GitHub Pages activé, les pages seront accessibles à :

- **Page d'accueil** : `https://samymas.github.io/tiktok-minecraft/`
- **Terms of Service** : `https://samymas.github.io/tiktok-minecraft/terms.html`
- **Privacy Policy** : `https://samymas.github.io/tiktok-minecraft/privacy.html`

## Activation de GitHub Pages

1. Allez dans **Settings** du repository
2. Allez dans **Pages** dans le menu de gauche
3. Sous **Source**, sélectionnez **Deploy from a branch**
4. Choisissez la branche **main** et le dossier **/docs**
5. Cliquez sur **Save**

Les pages seront automatiquement déployées via GitHub Actions.

## Mise à jour

Les pages sont automatiquement mises à jour à chaque push sur la branche main contenant des modifications dans le dossier `docs/`.
