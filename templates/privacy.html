Politique de confidentialité

Dernière mise à jour : [DATE]

1. Collecte de données

Cette application collecte uniquement les données nécessaires à l'intégration TikTok Live :

  • Événements de live : Cadeaux, donations, likes, commentaires, abonnements reçus pendant vos lives TikTok
  • Informations de base du compte : Nom d'utilisateur TikTok et ID unique (pour l'authentification via l'API TikTok)

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Recevoir les événements en temps réel pendant vos lives TikTok
  • Transmettre ces événements à votre instance Minecraft
  • Déclencher les actions correspondantes dans le jeu

3. Stockage des données

Les données sont traitées en temps réel et ne sont pas stockées de manière permanente. Aucune base de données n'est utilisée pour conserver vos données personnelles.

4. Partage des données

Nous ne partageons, ne vendons ni ne louons vos données personnelles à des tiers. Les données sont utilisées exclusivement pour le fonctionnement de l'application.

5. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données contre tout accès non autorisé, altération, divulgation ou destruction.

6. Vos droits

Conformément au RGPD et aux lois sur la protection des données, vous avez le droit de :

  • Accéder à vos données personnelles
  • Demander la rectification de vos données
  • Demander la suppression de vos données
  • Vous opposer au traitement de vos données

7. Cookies

Cette application n'utilise pas de cookies pour le suivi des utilisateurs.

8. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.

9. Contact

Pour toute question concernant cette politique de confidentialité ou pour exercer vos droits, veuillez nous contacter.
